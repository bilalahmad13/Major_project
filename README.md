# Daily Mood Logger

A simple C program to log your daily moods, view past entries, and search mood logs by date. This project is a console-based application that stores data in a text file (mood_data.txt).

---

## Features

- Add today's mood with a short note.
- View all mood logs.
- Search logs by a specific date.
- Simple console interface.
- Stores data persistently in a text file.

---

## Mood Options

When adding a mood, you can choose from:

1. Happy  
2. Sad  
3. Stressed  
4. Angry  
5. Neutral  

---

## How It Works

1. The program displays a menu with options:
   - Add Today's Mood
   - View All Logs
   - Search Logs by Date
   - Exit
2. When adding a mood, you enter the mood number and write a short note.
3. The program automatically adds the current date and stores the entry in mood_data.txt.
4. You can view all logs or search for a log by a specific date.
